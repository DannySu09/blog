transform的3d变幻

发表于 2014-09-17 | 分类于 CSS

当自己觉得有些东西在脑袋里躺得好好的呢，有啥忘的？这时，其实已经晚了。—— 佚名

我们在看这个世界，看2D是在看，看它在离我们50米开外一片幕布上变幻。甚至我们可以走到它跟前不到一米处看，因为你知道幕布永远和视线垂直，不会有东西能从那块幕布里蹦出来袭向我们的。其实，在2D的世界里，并没有一个「我们」的视角站在一旁观看，2D就是2D，幕布里的变幻，就是这个世界的全部了。

但是3D就不一样了。回想下电影贞子里鬼从电视里钻出来的画面。我们觉得很恐怖，是因为2D世界被打破了，开始有东西袭向我们了。又比如，2012年重新制作的3D版泰坦尼克号，为什么前排的同学情不自禁的向前摸去？因为……不说了，连卡梅隆都忍不住说了，那个3D的效果太逼真了。所以，3D就是多了一根坐标轴，方向和前排同学伸出去的手平行。
