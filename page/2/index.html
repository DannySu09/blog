书名：Not By Chance Alone
作者：Elloit Aronson
篇幅：320 页

  1. 前后花了差不多两个月
  2. 积累的词汇并没有统计
  3. 碰到的熟悉词挺多
  4. 自己读的太快，很多长句都没有理解，就为了量而敷衍过去了
  5. 语法生疏了（一直在生疏）

  • 2015-01-17
  • 说说

《佐藤可士和的超整理术》里就提过，太多东西分散注意力，该不带就不带。

  1. 前阵子，有那么个把星期，总是给自己安排很多东西。最后的结果却很搞笑，什么事情都没做成。闲时，我就开始想，在被很多东西拖着，却啥也没做的那段时间，我实际上在干嘛。发现，我一直在纠结，到底要干什么？好像先做哪件事都对不起我这宝贵的事件和其他那些在等着我做的事情。那些事情好像妃子一样，等着幸临，可惜翻牌无能。

    后来一想，其实是大脑在装。说白了就是啥也不想做，或者每做一件事，就会被剩下的事情不断提醒，你丫还有这么多事情，可时间就剩那么一点！心里就开始愧疚啊，感觉无力，为了掩盖自己的无能，大脑又开始在已列出来的长长清单下，又画上了那么几条……

了解 HTTP 协议
  • 2014-12-06
  • Web 基础

今天周六，睁眼看表，已经是11点30分，马刺的比赛错过了，舍友也已不见踪影，窗外淡淡橘黄的阳光让人觉得这还是早晨。

TCP & HTTP

打开 Google，搜索「HTTP tutorial」，看了几篇文章，老是出现 TCP 这个词，然后后面跟一句，HTTP 是应用层的协议，基于 TCP 这个传输层的协议。其实这个道理我一直都听过，但是也没弄清楚过到底是什么意思。

好了，让我们弄清楚他们。首先，什么「应用层」，什么「传输层」，先当他们是个名词吧。不要在心里默念「这特么到底是什么意思」，就只当是两个名词，就好像「苹果」和「香蕉」一样。
