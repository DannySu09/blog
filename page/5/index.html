transform的3d变幻
  • 2014-09-17
  • CSS

当自己觉得有些东西在脑袋里躺得好好的呢，有啥忘的？这时，其实已经晚了。—— 佚名

我们在看这个世界，看2D是在看，看它在离我们50米开外一片幕布上变幻。甚至我们可以走到它跟前不到一米处看，因为你知道幕布永远和视线垂直，不会有东西能从那块幕布里蹦出来袭向我们的。其实，在2D的世界里，并没有一个「我们」的视角站在一旁观看，2D就是2D，幕布里的变幻，就是这个世界的全部了。

但是3D就不一样了。回想下电影贞子里鬼从电视里钻出来的画面。我们觉得很恐怖，是因为2D世界被打破了，开始有东西袭向我们了。又比如，2012年重新制作的3D版泰坦尼克号，为什么前排的同学情不自禁的向前摸去？因为……不说了，连卡梅隆都忍不住说了，那个3D的效果太逼真了。所以，3D就是多了一根坐标轴，方向和前排同学伸出去的手平行。

读了《自控力》第四章，关于「明日复明日」的部分让人共鸣强烈。

在日常生活中，明日复明日的时候太多了。近期在我身上表现得比较明显的就是阅读任务。最常见的，今晚我想大概看二十页的书，可能会查15~20个单词。但是，下班回家，把心里觉得开始看书之前应该干完的事干完了，抬头一看已经快十二点了。如果坚持完成，那就要熬到一点了。

这时，有个声音就会冒出来说，前辈说过，「年轻人，没有什么工作是需要熬夜完成的」。

温习函数执行环境和作用域链
  • 2014-09-03
  • JavaScript

子曰：温故而知新，可以教妹子。

这几天写JavaScript写着写着，感觉越写越像一坨东西。这坨东西里面的东西怎么工作的，会造成什么问题，我几乎一无所知。我所能知道的，就是刷新页面时，它好像能做我想让它做的事情。

好像是在知乎上，看到一句话：优秀的程序员和普通的程序员最大的区别是什么？优秀的程序员对自己在做的事情了如指掌。

这句话本意也许是，优秀程序员不仅对写下的代码了如指掌，而且还对自己写下的服务于的目标领域，也了如指掌。
