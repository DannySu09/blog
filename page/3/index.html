TCP & HTTP

打开 Google，搜索「HTTP tutorial」，看了几篇文章，老是出现 TCP 这个词，然后后面跟一句，HTTP 是应用层的协议，基于 TCP 这个传输层的协议。其实这个道理我一直都听过，但是也没弄清楚过到底是什么意思。

好了，让我们弄清楚他们。首先，什么「应用层」，什么「传输层」，先当他们是个名词吧。不要在心里默念「这特么到底是什么意思」，就只当是两个名词，就好像「苹果」和「香蕉」一样。

Node

Node是最上层的一种类型。所以，Document（文档）是一种Node，Element（元素）也是一种Node。作为Node类型的实例，它们都有一些共同的属性，比如nodeType、nodeName、nodeValue，parentNode等等。点击查看更详细的属性

window.define的写法导致mismatch错误

第一次用RequireJS，今天把所有模块及其依赖用r.js打包后，RequireJS一直在抛mismatched的错误。盯着打包后的代码看了一会儿，终于找到了问题。

假设有一个模块叫做Countdown，返回的是变量名为Countdown的构造函数，然后就有了如下的代码：

    
    if(window.define != null) {
      window.define(function(){
          return Countdown;
      });
