微信小程序开发日志

  • 2016-09-24
  • JavaScript
  • 微信小程序 

这两天都在赶工公司的小程序，踩了一些坑，总结如下：

  • 在根目录下的 app.js 中不能 require，会导致报 Page 找不到的错误。
  • UI上尽量不用 fixed 定位，当键盘弹出时，fixed 定位的元素不能正确布局，有点类似于当初 iOS 8 中的 Safari 不能正确布局 fixed 定位的感觉。；navbar 也不能正常显示，会显示成一片白。
  • audio 组件接收一个 action 属性，可以通过设置这个传入的属性来控制播放、暂停、调到指定时间等操作。
  • input 组件可以在 wxss 中直接定制样式。
