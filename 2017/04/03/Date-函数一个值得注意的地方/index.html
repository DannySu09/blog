Date 函数一个值得注意的地方

Apr 3, 2017

到了 3 月的 31 号的时候，如果你直接调用：

var d = new Date()
d.setMonth(d.getMonth() + 1)

console.log(d.getMonth() + 1)

执行上面的代码会发现打出来的是 5。

原因

其实 setMonth 还接受第二个 dayValue 参数，在 MDN 描述如下：

If you do not specify the dayValue parameter, the value returned from the getDate() method is used.

这会导致如果下个月只有30天，就变成了 setDate 为 31 日，超出了一天，月份就会自动加 1，跳到下一个月去了。
