微信小程序开发日志

发表于 2016-09-25 | 分类于 JavaScript

这两天都在赶工公司的小程序，踩了一些坑，总结如下：

  • 在根目录下的 app.js 中不能 require，会导致报 Page 找不到的错误。
  • UI上尽量不用 fixed 定位，当键盘弹出时，fixed 定位的元素不能正确布局，有点类似于当初 iOS 8 中的 Safari 不能正确布局 fixed 定位的感觉。；navbar 也不能正常显示，会显示成一片白。
  • audio 组件接收一个 action 属性，可以通过设置这个传入的属性来控制播放、暂停、调到指定时间等操作。
  • input 组件可以在 wxss 中直接定制样式。

webpack 如何 shim 模块

发表于 2016-04-09 | 分类于 JavaScript

看了 webpack 的官方文档，还是稀里糊涂的，决定imports-loader exports-loader 挨个研究。明白了两个 loader 的各自作用后，果然就明白了如何去 shim 一个模块。

imports-loader的作用是，当要引入一个模块时，在引入之前，先把一些赋值语句注入到那个模块中，然后再引入。

exports-loader的作用是，当要引入一个模块时，在引入之前，先把一些导出模块的语句加到那个模块中。也就是commonjs的module.exports = xxx或者exports.a=xxx这类语句。
