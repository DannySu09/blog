简单记录下 2014

Jan 5, 2015

2014 年于我最重要的一件事，就是学生生涯结束了。
那么……人生就此开启新篇章？
不是的，昏暗的宿舍里电脑屏幕的光照不透自己的窟窿。
出来才发现满身是洞，得补。

改善体质，开始坚持锻炼，无氧和有氧结合着来。
发现各种人情世故都变复杂了，自己的情绪也会变得糟糕，开始尝试观察和调理情绪。
但有时，还是老觉得谁谁谁很傻逼。现在越来越觉得，每个人都有不容易的地方。不管前后就觉得别人傻逼的自己才傻逼。
开始接触很多 JavaScript 的新玩意儿。
但总觉得我的 CSS 从来没有进步过。
HTML 也是水得很。
还曾经想过好好学设计，摆弄了一段时间的 Sketch，画了点东西。
偷偷的跟自己说过，要多久多久之后成为一名合格的前端工程师。
然后发现，一直在贪心的东看西看，很多基础的东西尚未牢固，就想着要学这个学那个。有时会疯狂的学一些新东西来获得进步的虚幻感。

继续好好调理情绪。
继续巩固基础。
珍惜身边的朋友，更多的站在他们的角度想想。
珍惜身边的亲人，好好的爱爸爸妈妈。
