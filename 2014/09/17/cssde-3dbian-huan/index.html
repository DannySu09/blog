transform的3d变幻

发表于 2014-09-17 | 分类于 CSS

当自己觉得有些东西在脑袋里躺得好好的呢，有啥忘的？这时，其实已经晚了。—— 佚名

我们在看这个世界，看2D是在看，看它在离我们50米开外一片幕布上变幻。甚至我们可以走到它跟前不到一米处看，因为你知道幕布永远和视线垂直，不会有东西能从那块幕布里蹦出来袭向我们的。其实，在2D的世界里，并没有一个「我们」的视角站在一旁观看，2D就是2D，幕布里的变幻，就是这个世界的全部了。

但是3D就不一样了。回想下电影贞子里鬼从电视里钻出来的画面。我们觉得很恐怖，是因为2D世界被打破了，开始有东西袭向我们了。又比如，2012年重新制作的3D版泰坦尼克号，为什么前排的同学情不自禁的向前摸去？因为……不说了，连卡梅隆都忍不住说了，那个3D的效果太逼真了。所以，3D就是多了一根坐标轴，方向和前排同学伸出去的手平行。

所以，在3D的世界里，距离就很重要了。假设，贞子向我们扑来，而我们站在不懂。贞子离我们十米时，我们会觉得寒气逼人；离我们五米远，会惊讶鬼居然有口臭；离0米远，我们就光荣牺牲了。

先介绍几个关键属性：

  • rotateX
  • rotateY
  • perspective
  • translateZ
  • backface-visibility

##perspective

我们站在电视机前10米处看贞子，这时，perspective的属性值就是10米。当然这只是个比方，在CSS里，这会用px来代替。

perspective其实就是指，某物体在做3D变幻时的轴心。比如，当在踩自行车时，一双脚踏在做运动，轴心是中间的轴承。当卫星在绕地球运动，地球球心是轴心。当单杠运动员在单杠上旋转，单杠是轴心。当我们在看别人在跳钢杆舞，那钢杆就是轴心。

这时，应该很清楚perspective的含义了。没错，就是你的眼睛到钢杆的垂直的距离。也就是你在看着3D变幻时，离物体运动中心的距离。那如果想离近点看呢？没错，就是修改perspective的大小。但如果是看钢杆舞女，走近点看有点猥琐。如果不是主动走向前去，而是钢杆舞女主动靠近，那就不是猥琐，而是气质迷人了。

早前不是有人说过吗？

一个艺校女生虽然天天晚上自习，但是深夜就去夜总会陪酒，这就是不爱惜自己。

另外一个艺校女生，虽然天天晚上去夜总会陪酒，但是仍坚持天天晚上自习，这就是励志。

同理

你主动靠近钢杆舞女去看，这叫猥琐。

你站在原地，等着钢杆舞女靠近你，这叫气质宏伟。

那有什么办法能让我们气质宏伟？引出第二个属性。

translateZ

主要作用是，改变物体围绕轴心做运动时的运动半径。

比如，你悬浮在太空中，离地球1光年的地方，看着一颗离地心0.5光年的卫星做无聊的绕地球旋转运动。突然，地球上爆发了第三次世界大战，原子弹把地心炸残了一半，地球的万有引力也残了一半，卫星得以做离心运动，最终在离地心0.8光年的地方稳定下来，继续做绕地球运动。

这时，就相当于translateZ的值从0.5光年，变到了0.8光年。

钢杆舞女同理，钢杆舞女大腿绕着钢杆跳舞，这时可认为translateZ是0，但是当她被你吸引，走向了到了你跟前，假设之前你离她十米，那现在translateZ的值就是10米。懂了吧？

但是，如果钢杆女继续在translateZ这个值上做运动，那她就得绕着钢杆，坐半径为10米的运动……好累啊，感觉不会再跳了。

这时，钢杆是竖着的，我们可以把它当做Y轴。你说绕钢杆运动，那如果女郎刚好转到理我远的那一端呢，translateZ变为10米不是反而离我更远了吗？

那把她转过来不就离得近了嘛。

那你倒说怎么转啊？？

祭出第三个属性。

##rotateY

看到这个属性的这一刻，如果你会用2D方法中的rotate方法，相信凭直觉，也能猜出七八分来这玩意儿怎么用。我目前只会低端用法，那就是直接给它设角度值。以舞女的身体垂直你的身体时为0deg（好销魂的比喻），舞女离你远时，其实是转到了90deg的位置，那离你近是呢？自然是-90deg或者270deg。

##rotateX

看到这个属性的那一刹那，你肯定在想舞女变身单杠运动员，钢杆由竖变横的情境。没错，就是这么回事儿。那不用我说，你大概也能知道了，这就是绕着X轴做运动。我也只会低端玩法，直接赋角度值。

##backface-visibility

假设舞女做绕Y轴运动呢，0deg时，面朝你，180deg时呢？背朝你。backface是啥意思，visibility是啥意思？合起来是啥意思？在背后能不能看到的意思。想象下一份墨宝，大笔一挥写下的C字母，本来开口向右，到了180deg这个角度时，我们从纸背后透过一点点伸出来的墨迹看，是不是开口向左了？

所以，这个属性的意思就是，正面你能看到了，那你还能不能从背后再透视过去再看到呢？我就不举舞女的例子了，太猥琐了。不过你应该懂我的意思。

就谈到这儿吧。
